        Parametre: api kluc, nazov funkcie - "airplane", parametre pre CAS
        Odozva z API: { cas[],naklon lietadla[], naklon zadnej klapky[] }

Statistika (GET/POST/MAIL)
    Popis (GET): vrati json s hodnotami z databazy navstevnosti stranok
        Parametre: api kluc, nazov funkcie - "getStat"
        Odozva z API: { nazov stranky: pocet, ... }
    Popis (POST): aktualizuje statistiku stranok v databaze
        Parametre: api kluc, nazov funkcie - "incStat", parameter pre funkciu(nazov stranky)
        Odozva z API: { true - ak vsetko prebehlo vporiadku }
    Popis (MAIL): odosle mail so statistikami na zadanu mailovu adresu
        Parametre: api kluc, nazov funkcie - "sendMail", parametre pre funkciu(mailova adresa)

Konzola (GET)
    Popis: vrati json s vysledkom pre zadany prikaz ktory vyrata CAS
        Parametre: api kluc, nazov funkcie - "cmd", parameter - prikaz do CAS
        Odozva z API: { vysledok z CAS pre zadany prikaz }
